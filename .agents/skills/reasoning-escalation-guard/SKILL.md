# Reasoning Escalation Guard

- Codex が自動で推論深度を変更するとは書かない。
- 曖昧、複雑、高リスク、複数ファイル、設計判断、データ影響、セキュリティ、権限、外部副作用がある場合は停止する。
- 停止時は `gpt-5.5 xhigh` 推奨として、理由、確認済み内容、未確認事項を出す。
- 切替後に貼る続行プロンプトを提示する。
- 安全に限定できる 1 checkpoint の場合だけ、通常の範囲で続行する。
