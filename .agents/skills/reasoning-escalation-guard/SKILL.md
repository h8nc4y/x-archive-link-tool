# Reasoning Escalation Guard

- Codex が自動で推論深度を変更するとは書かない。
- 曖昧、複雑、高リスク、複数ファイル、設計判断、データ影響、セキュリティ、権限、外部副作用がある場合は、対象範囲、仮定、検証方法、停止条件を明確にして進める。
- 複雑さや高リスクであることだけを停止理由にしない。料金発生、paid service、secret/credential/実データ外部送信、sandbox/permission/usage-limit blocker がある場合だけ停止する。
- ローカルで限定できる作業は、小さな差分と検証単位に分けて継続する。
- 停止が必要な場合は、理由、確認済み内容、未確認事項、代替案、承認後の安全な続行手順を提示する。
